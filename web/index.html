청소 모듈 디테일

6가지 카테고리 자동 분류

  • 거대 파일

    50MB 이상의 단일 파일. 보통 영상·iso·설치 파일.

  • 오래 묵은 파일

    마지막 수정 90일이 지난, 잊고 있던 파일들.

  • 스크린샷 잔해

    Screenshot_*, 스크린샷 *, Screen Shot * 등 패턴 매칭.

  • 임시·잔해 파일

    .tmp, .crdownload, .part, .bak, ~$… 같은 임시 파일과 0KB 파일.

  • 중복 파일

    SHA1 해시로 동일 파일 탐지. 가장 오래된 1개는 자동 보존 추천.

  • 빈 폴더

    내용물 없이 자리만 차지하던 폴더.

동작 방식

4단계로 끝납니다

  1. 01 스캔할 폴더를 지정. 기본: 바탕화면 · Downloads · Documents.
  2. 02 카테고리별 회수 가능 용량을 보여줍니다.
  3. 03 안전한 항목 자동 선택, 또는 직접 체크박스로 고름.
  4. 04 "휴지통으로 보내기" 한 번. 복원 가능합니다.
